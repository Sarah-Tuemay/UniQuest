# UniQuest 🎓💼
A campus job marketplace connecting Ethiopian university students with flexible on-campus work.

## Stack
- **Frontend** — HTML, CSS, Vanilla JS, React 18 (jobs search)
- **Backend** — PHP 8+, MySQL

## Structure
```
UniQuest/
├── index.html          ← entry point
├── frontend/           ← pages, css, js, images
└── backend/
    ├── api/            ← auth.php, profile.php, apply.php, me.php
    ├── database/       ← schema.sql
    └── uploads/
```

## Setup
1. Drop project into XAMPP/WAMP `htdocs`
2. Run `backend/database/schema.sql` in phpMyAdmin
3. Update credentials in `backend/api/config.php`
4. Open `http://localhost/UniQuest/`

> No database? The site runs in demo mode automatically.
